# Changelog

Все значимые изменения проекта. Формат — [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/), версионирование — [SemVer](https://semver.org/lang/ru/).

## [0.3.1] — 2026-06-10

Bugfix-релиз по итогам live smoke-теста против реального портала
(план: `docs/superpowers/plans/2026-06-10-live-smoke-bugfixes.md`).

### Исправлено

- **`gramax_search` был полностью неработоспособен на реальном API** (#3):
  `breadcrumbs[].title` приходит списком фрагментов `{type, text}`, а не строкой
  (падал `TypeError`); у `properties` ключ `id`, а не `name` (падал `KeyError`).
  Добавлена контрактная фикстура по реальному ответу портала.
- **5xx и сетевые ошибки утекали англоязычным traceback** (#5): добавлены
  `GramaxServerError` и `GramaxNetworkError`, все HTTP/сетевые/JSON-ошибки
  клиента теперь возвращаются русскими сообщениями (конвенция проекта).
- **Пустой `GRAMAX_API_TOKEN` ронял httpx** (`Illegal header value`) (#4):
  токен стал опциональным — без него сервер работает в анонимном режиме
  для публичных порталов (ADR-0002). Обратносовместимо: с валидным токеном
  поведение не меняется.

### Известные ограничения

- `GET /api/user/token` требует браузерную сессию — автообновления токена нет;
  просроченный токен даёт 401 даже на публичных эндпоинтах (уберите его из
  `.env`, чтобы работать анонимно).
